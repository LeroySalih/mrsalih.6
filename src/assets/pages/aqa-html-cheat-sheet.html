AQA iGCSE HTML Cheat Sheet.

The following HTML Cheat Sheet core requirements for the AQA iGCSE course in Computer Science. For a complete web reference, you should visit www.w3schools.com.
Document Structure
The following HTML demonstrates the standard structure of a HTML document.

        <!-- Tell the client browser that this document is HTML-->
        <!DOCTYPE HTML >
        
        <!-- Start of the HTML Document -->
        <HTML>

          <HEAD>
            <!-- Document info goes here -->
            <!-- Stylesheet links go here -->
            <!-- Script links go here -->
          </HEAD>

          <BODY>
            <!-- Body content goes here -->
            <!-- Script to run when page loads goes here -->
            
          </BODY>

        </HTML>
Block Level Tags
The following tags are BLOCK LEVEL tags. This means that they apply to the whole section.
        Tags
        <DIV></DIV>
        DIV blocks mark out sections of the page. A DIV can contain other DIV or P elements.
      
        <P></P>
        A P marks a paragraph. It can not contain other DIV or P elements.
      
        <Hx></Hx>
        H1..H6 header elements, mark a section as being a Page Header. 
      
        <UL></UL>, <OL></OL>, <LI></LI>
        UL, OL combine with LI tags to create Ordered or Unordered Lists. 
      

        Attributes
        id=""
        style=""
Inline Level Tags
The following tags are Inline LEVEL tags. This means that they apply to the whole section.
        Tags
        <STRONG></STRONG>
        STRONG tag mark out sections of text that are to be emphasised.  Most browsers display using bold.
      
        <SPAN></SPAN>
        A SPAN marks section of text in a paragraph. It can be used to apply styles to specific areas of the page.
      
        <I></I>
        Mark a section of text as using Italic.
      
        <U></U>
        Underline a section of text.
      

        <BR/>
        Create a break in the page. This can be used to add space after a paragraph, or to breakup a line of text. 
      
        <HR/>
         Creates a break in a page, that is underlined. 
      
        Attributes
        id=""
        style=""
Image Tag
The following tags are Inline LEVEL tags. This means that they apply to the whole section.
        Tags
        <IMG></IMG>
        DIV blocks mark out sections of the page. A DIV can contain other DIV or P elements.
      
        Attributes
        src=""
      
        Examples
        <IMG 
          scr="http://www.mrsalih.org/assets/images/mrsalih_logo.png" 
          style="width: 50px; height: 30px;" >
        </IMG>
Link Tag
The following tags are Link tags. This means that they apply to the whole section.
        Tags
        <A></A>
        The Anchor tag (A) creates a link to either another section on the current page or a different page.
      
        Attributes
        href=""
        This is the URL or bookmark the link will take the user to.

        target=""
        Used to specify whether to open the link in the same browser window (_self) or a new browser window (_blank) 

        Example
        <A  href="www.bbc.com"  target="_blank" >The BBC Web Site.</A>
        
        produces

        The BBC Web Site.
