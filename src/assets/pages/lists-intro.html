Background.

At this point, we are comfortable with the idea that we can store data in variables. We should also be comfortable with the idea that a variable has:
  • A Name
  • A Data Type
  • A Value
So, lets imagine that we want to store the names of pupils in a class. Our first approach may be to store each pupil name in a variable.
However, this is going to lead us to a number of questions, such as:
  • How many pupils are there in the class?
  • How many variables do I need to declare?
  • What happens if pupils join or leave the class?
So, what we need is a flexible way to store pupil names. A way that will allow us to add or remove names if pupils join or leave the class. This is what lists provide.

Lists

A Python List allows us to create a list of variables. We can then think of this list as either a single variable (with a data type of list) or as a collection of variables (with assorted data types).
Just like any variable, we need to declare a list before we use it.

List Indexes

Once we have declared our list, we can either treat the list as a single variable, or access individual items within the last. To access a single item, we use the [ ] operator. Between the [ ], we place the position of the item we want to access. It is important to note that lists start counting at 0. The first item is [0]. We can use [ ] to either read or write to the list.

Adding Items.

We have seen that we are able to initialise a list using code similar to:

myList = ['Albert', 'Brian', 'Charles']

We can also add or remove items to lists while the program is running. We can use the append() function to add an item to the end of a list. This is often used in collecting data from the user.
The append() function will place the new item at the end of the list. We can also use the insert() function to place a new item at any position in the list.

Removing Items

When we need to remove an item from the list, we have 4 options.
  • pop() We can call pop() to remove the last item of the list.
  • remove() We can call remove() remove a specific item from the list.
  • del We can use del to remove a specific position from the list
  • clear Will remove all the items from the list.
