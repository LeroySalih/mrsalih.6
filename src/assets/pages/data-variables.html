Background.

A computer program is a combination of COMMANDS and DATA. Data is the information that flows around the computer program, while COMMANDS are used to inputinput, output and process the data.
 
When we need to remember a value, we can ask the computer to store the value in memory. A computer will store the value in a Variable.
 
A variable can be thought of as a box in computer memory. When the computer needs to remember a value, the value is placed in the box. The value can be retrieved from the box at a later time by the computer.
Before we can use a variable, we must:
  • Tell the computer that it needs to reserve a box. This is called "declaring" the variable.
  • Once the variable has been declared, it must have a value placed in it. This is called "Initialising" the variable.
  • Once the variable has been declared and initialised, it can be "used" by the commands of the program.

Properties of Variables

We have said that we can think of a variable as a box, where we store data. We must also be aware of the properties of a variable. All variables have the following properties:
  • A Name.
  • A Data Type.
  • A Value.

Variable Names

The Name of the variable is used to identify which variable we want to use in our program. A program may have 1, 10 or a 1,000,000 variables, so it is important to be able to identify specific variables when we want to use them in calculations. Each programming language has different rules for what you can call a variable. In python, a variable name:
  • Can not contain a space or punctuation character (other than an underscore '_')
  • Can not start with a number
Also, variable names should be Meaningful. This means that you should be able to understand the purpose of a variable from its name.

Variable Data Types

The Data Type of a variable determines which values a variable can hold. Although there are many Data Types, the main ones are:
  • integer - can hold any whole number
  • real or float - can hold any decimal number
  • string - can hold any sequence of characters
  • boolean - can hold either True or False
Some programming languages require to declare the data type at the same time as declaring the variable name. However, python will guess the Data Type of a variable based on the value. Also, the data type can change if the value in the variable changes.
We can use the type command to check the Data Type of a variable.
