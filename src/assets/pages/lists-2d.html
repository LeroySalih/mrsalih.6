Background.

We have seen that we can collect many variables together and store them in a python list.
In the previous page, we used lists to store and process strings and integers. However, we can also create lists of lists. This technique is called 2Dimensional (2D) lists.
We can declare 2D lists in two ways:
  • Declare each list, then gather them into a new list (exampel 1 below).
  • Declare a new list, that already contains other lists (example 2 below)

Accessing Items in 2D lists

When we are dealing with a 2D list, we can still use the [ ] operators to access items. However, we need to use one set of [ ] for each dimension.
In our 2D list, we will use 2 sets of [ ] to access the required item.

Adding items in 2D lists

To add items, we can use the [ ] operator to access the desired row. Then we can use the usual functions to add, remove, sort or count items.
