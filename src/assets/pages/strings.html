Background.

A string is a collection characters, and is one of the key data types that every program will use.
A string can be identified by the quotes that are used to declare the variable (either ' or " ).

Declaring String Variables.

To declare a string variable, you use one of the two types of quotes ( e.g. 'Hello' or "World").
It is also possible to use a string as a constant value, as in the case of printing a simple message.

Comparing String Variables.

There are a number of comparision operators we can use with strings. The following boolean operators will either return True or False.

  • The == operator can be used to determine if two strings are equal. This means that they contain exactly the same characters, in the same order.
  • The != operator can be used to determine if two strings are not equal.
  • The > operator can be used to determne if one string is 'greater than' the other. A string is greater than another string if it is further in the alphabet. For example, 'B' > 'A' == True

String Operators and Functions.

There are a number of other very useful string operators and functions. Please note: a more detailed list can be found at TutorialsPoint.com
  • len() will return the length (number of characters) of a string.
  • find() will return a index of a string if appears in another string. If it does not appear, it will return -1
  • upper(), lower() and title() will return a string that is formatted in either Upper Case, Lower Case or Title Case.
You can also the following operators:
  • str1 + str2 This will return the 2 strings joined.
  • str1 * 3 This will return the string repeated 3 times.
There are many other functions that be used to manipulate strings. Please visit TutorialsPoint.com for more information.
