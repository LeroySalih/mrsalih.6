Background.

It is often required to examine each item of an array. For instance, to find a name in a list of pupils, or generate a sum of all exam scores. Looping through all the items of a list is known as Iteration.

There are many mechanisms we can use yto iterate a list, in this session, we will discuss:

  • Using in operator
  • Using [] and for loop
  • Using [] and a while loop

Using in operator

We have already used the in opeator, when we initialise our for loops. We often see code such as

for index in range(0, 10, 1):

If we wish to iterate through our loops, we can use the same approach (as in example 1 below).

However, there is a gotcha to be aware of! When we access items usign the in operator, we are not accessing the actual item, but a copy of the item. This means that if you make changes to the item, they are not reflecting in the original list. To overcome this issue, you can use the enumerate command (as in exmaple 2 below).
