Command Categories.
The programming concepts are computer commands that are grouped into 3 groups; Sequence, Selection and Iteration.
  • Sequence.These commands are executed one after another (i.e. in sequence). For example, the sequence commands for a robot may be:
    • GO LEFT
    • GO FORWARD 100 STEPS
    • STOP
  • Selection.These commands tell the computer to chose between 2 or more options (i.e. select). For example, a selection command for a robot may be:
    • IF it is hot outside
    • THEN stay inside
    • ELSE go outside
  • Iteration.These commands tell the computer to repeat other commands either 0, 1, More than 1 or keep going until something happens. For example, an iteration command for a robot may be:
    • WHILE the car is dirty
    • CLEAN CAR
